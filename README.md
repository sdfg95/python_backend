# Домашняя работа №1

## Шаги по запуску

### При помощи Makefile
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/sdfg95/python_backend.git
   cd python_backend
2. Запустите файл
    ```bash
   make -f Makefile.mk

### При помощи run.sh/run.bat
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/sdfg95/python_backend.git
   cd python_backend
2. Запустите файл на мак или Линукс
    ```bash
   chmod +x run.sh
   ./run.sh
3. Запустите файл на windows
   ```bash
   run.bat
